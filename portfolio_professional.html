Computational approaches to decode neural mechanisms of behavior

Neural Activity Decoding of Exploratory Behavior

Michigan Neuroscience Institute â€¢ 2020-2024 â€¢ Published: eNeuro 2022

Project Overview

Developed AI/ML pipelines to decode behavioral states from hippocampal CA1 neural activity during object exploration in freely-moving mice. Combined in vivo calcium imaging (1-photon, Inscopix) with behavioral tracking to investigate how stress hormone receptors alter neural encoding patterns.

This project required building end-to-end computational infrastructure from raw imaging data processing through advanced neural decoding models, resulting in novel insights into how glucocorticoid signaling modulates hippocampal representations of exploration.

Key Achievements

  • Achieved 84.7% accuracy in decoding exploratory states using neural activity, improving upon baseline by 25%
  • Implemented time-resolved decoding framework revealing temporal dynamics of neural population coding
  • Developed causal window analysis demonstrating that recent neural history predicts current exploration state
  • Created reproducible analysis pipeline processing 100+ hours of simultaneous neural-behavioral data
  • Integrated molecular (RNA-seq), neural (calcium imaging), and behavioral datasets to identify multi-level mechanisms

Computational Methods & Pipeline

Image Processing Pipeline

Motion correction (NoRMCorre), source extraction (CNMF-E), and deconvolution (OASIS) algorithms for calcium imaging cell registration and calcium trace extraction from raw video data

Behavioral Analysis

Automated tracking and classification of exploration periods with frame-by-frame synchronization to neural data using custom Python pipelines

Machine Learning Models

Compared classical (Logistic Regression, SVM, Random Forest) vs. deep learning approaches (MLP, GRU) for neural decoding, with ROC/AUC performance evaluation

Temporal Decoding Framework

Implemented instantaneous, causal window, and non-causal window decoding to assess temporal integration properties of neural population codes

Research Findings

Time-Resolved Exploration Decoding

Instantaneous frame-by-frame decoding shows that neural population activity reliably predicts exploration states throughout behavioral sessions, with model predictions aligned to true exploration periods.

Temporal Integration Across Decoding Windows

Accuracy increases from 56.8% (instantaneous) to 78.6% (causal window) to 84.7% (non-causal window), revealing that neural ensembles integrate information across time to encode exploration.

Classical vs. Deep Learning Decoders

ROC analysis comparing multiple algorithms shows that deep learning models (GRU: AUC=0.831, MLP: AUC=0.752) outperform classical approaches (Logistic Regression: AUC=0.738), demonstrating value of neural networks for complex temporal patterns.

Calcium Imaging Processing Pipeline

End-to-end workflow from raw calcium imaging video through motion correction, source separation, and calcium trace extraction, ensuring high-quality neural activity data for downstream analysis.
