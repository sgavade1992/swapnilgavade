HYROX Training Hub
12-Week Program
Phase 1 — Aerobic Foundation
Weeks 1–4
Build Zone 2 aerobic base. Weekly mileage 8→14 miles. Introduce strength fundamentals. Sunday = full body strength + 15–20 min easy run.
Phase 2 — Threshold Development
Weeks 5–8
Add tempo runs, HYROX station combos, SkiErg work. Race-weight station training begins. Sunday = lower strength + SkiErg + easy run.
Phase 3 — Race Simulation
Weeks 9–12
Zone 4 intervals, half and full HYROX simulations, taper in Week 12. Sunday = strength + full station block + easy run.
Sunday Structure (Every Week)
Wks 1–4Full body strength (squat, RDL, push, pull) + 15–20 min Zone 2 run
Wks 5–8Lower strength + SkiErg intervals + 15–20 min Zone 2 run
Wks 9–11Strength + full HYROX station block + 20 min Zone 2 run
Week 12Light taper strength + 15 min easy run only
